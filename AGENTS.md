# Agent 使用说明

这是一个无代码 Agent Skill。不要寻找 scripts 目录；本 Skill 的主要能力来自结构化对话、参考文档和输出模板。

使用时请优先读取：

1. `SKILL.md`
2. 与任务相关的 `references/` 文件
3. 需要输出报告时读取 `assets/inspiration-distillation-report-template.md`
4. 需要落地实验时读取 `assets/seven-day-experiment-template.md`

重要规则：

- 不鼓励饮酒。
- 不做心理诊断。
- 不诱导隐私暴露。
- 先保护野性，再谈落地。
- 每次输出至少包含一个 7 天实验。
